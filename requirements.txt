# 👗 AI Fashion Try-On App

An AI-powered fashion try-on app built using:

- Python
- Gradio
- Stable Diffusion
- Groq LLM

## 🚀 How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Add your Groq API key in app.py

3. Run:
   python app.py
